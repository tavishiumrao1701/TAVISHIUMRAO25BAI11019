# 💳 Loan Approval Prediction AI

## 📌 Project Overview

This project uses Machine Learning to predict whether a loan application will be approved or rejected based on financial factors.

The model analyzes:

- Income
- Credit Score
- Loan Amount

and predicts:

- 1 = Loan Approved
- 0 = Loan Rejected

---

## 🚀 Technologies Used

- Python
- Pandas
- Scikit-learn
- NumPy
- Machine Learning (Logistic Regression)
